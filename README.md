> # Git 使用
> ```shell script
> git init # 初始化本地仓库
> git add . # 添加当前目录文件
> git state # 查看状态
> git commit -m "备注" # 提交到本地仓库
> git push origin master # 推送到名为origin的远程仓库的master分支
> git pull origin master # 拉取名为origin的远程仓库的master分支
> ```
